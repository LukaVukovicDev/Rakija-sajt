# 🥃 Brandy Boutique - Rakija sajt

**Brandy Boutique** is a modern, responsive mini e-commerce website dedicated to fine brandy.  
It features a stylish landing page, product showcase, shopping cart simulation, and contact form — all built using **HTML**, **CSS**, **JavaScript**, and **PHP**.

## 🎨 Features

- **Landing Page** – elegant hero section introducing the brand and products  
- **Product Catalog** – dynamically loaded brandy bottles with images, prices, and descriptions  
- **Add to Cart Simulation** – basic cart logic using JavaScript  
- **Contact Form** – PHP-based backend form that sends messages or stores them  
- **Responsive Design** – optimized for desktop, tablet, and mobile screens  
- **Animations** – subtle hover and scroll animations using CSS & JS
